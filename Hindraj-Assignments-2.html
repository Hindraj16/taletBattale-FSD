Text Formatting Tags

HTML (Hypertext Markup Language) is the standard language used to create web pages. It is a markup language, which
means it uses tags and syntax to define the structure and layout of a webpage. HTML is used in conjunction with
other languages such as CSS (for styling) and JavaScript (for adding interactivity) to create a complete website.

HTML elements are represented by tags, which are surrounded by angle brackets (< >). Each tag has a specific
purpose, such as defining the title of a webpage or formatting text. HTML also uses attributes, which provide
additional information about an element, such as the link to a website webpage or the color of a piece of text.

HTML was first introduced in 1993 and has since become the standard language for creating web pages. It is used by
millions of websites around the world and is an essential skill for anyone looking to create their own website or
contribute to the web as a developer, designer or content creator.
